# SmartCar AI Platform 🚗🤖

Avtomobil ijarasi uchun sun'iy intellektga asoslangan platforma (Diplom ishi uchun).

## 🌟 Imkoniyatlar
- **Dinamik Narxlash (AI):** Talab va taklif asosida narxlarni avtomatik rejalashtirish.
- **AI Chatbot:** Mijozlarga mashina tanlashda va qoidalarni tushuntirishda yordam beradi.
- **Bashoratli Xizmat (Predictive Maintenance):** Mashinaning holatini tahlil qilib xizmat ko'rsatish ehtiyojini oldindan aniqlaydi.
- **Zamonaviy UI/UX:** Foydalanuvchilarga qulay va tezkor interfeys.

## 🛠 Texnologiyalar
- **Frontend:** Next.js, React, Tailwind CSS, TypeScript
- **Backend:** FastAPI, Python, Uvicorn
- **Ma'lumotlar Bazasi:** PostgreSQL, MongoDB
- **Boshqa:** Docker, Git

## 🚀 O'rnatish

1. Repozitoriyni yuklab oling:
```bash
git clone https://github.com/USERNAME/smart-car-ai-platform.git
cd smart-car-ai-platform
```

2. Muhit o'zgaruvchilarini sozlang:
`.env.example` faylidan nusxa olib `.env` deb nomlang va ichini to'ldiring.

3. Docker yordamida ishga tushiring:
```bash
docker-compose up -d --build
```

Platforma quyidagi manzilda ishga tushadi:
- Frontend: `http://localhost:3000`
- Backend API: `http://localhost:8000`
